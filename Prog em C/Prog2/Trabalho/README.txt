Para a execução do algoritmo, deve inserir como entrada onde o primeiro valor
é o tamanho da quantidade de elementos, e em seguida os elementos. Após isso 
deve selecionar o metodo de orddenação o qual pretende ultilizar, sendo eles: 
1 para inserção, 2 para merge, 3 para quick sort, 4 para heap sort e 0 para sair. 
Após isso será executada a ordenação e irá aparecer o resultado do tempo de execução.
